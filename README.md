# Image Battle

Small single-file image comparison tool.

## Use

Clone this repo, open `index.html` in a browser.

## Features

- Upload images incrementally
- Compare a baseline image against a candidate
- Side-by-side and overlay modes
- Independent or locked zoom/position controls
- Drag to line images up
- Reset view for the active compare pair
- Expand the compare area for closer inspection
